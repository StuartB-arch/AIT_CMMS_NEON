AIT CMMS APPLICATION SETUP
==========================

SYSTEM REQUIREMENTS:
- Windows 10 or later
- Python 3.8 or newer
- OneDrive for Business installed

INSTALLATION:
1. Ensure this folder is synced with SharePoint via OneDrive
2. Install Python from https://www.python.org/downloads/
3. Install required packages: pip install -r requirements.txt
4. Run Start_Application.bat to launch the application

REQUIRED PACKAGES (in requirements.txt):
pandas==1.3.3
tkinter==8.6
reportlab==3.6.8
